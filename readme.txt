A widget to display thumbnails and titles of the latest DOGOnews, DOGObooks, and DOGOmovies reviews via RSS.

== Description ==

This plugin allows you to place 3 different widgets on your sidebar that fetches the most recent contents from DOGOnews.com, DOGObooks.com, and DOGOmovies.com via RSS feed and displays the corresponding thumbnail images, titles, and descriptions. You can choose whether to show the description below the image, and you change the style of the widget to fit your theme.
Expose students to current events from DOGOnews.com, the leading source of content for Common Core State Standards ELA, science and social studies.  Share latest book reviews from DOGObooks.com, where kids review and rate books.  And for some fun, share movie reviews by kids from DOGOmovies.com.

== Installation ==

1. Upload the folder `dogo-rss-widget` and its contents to the `/wp-content/plugins/` directory or use the wordpress plugin installer
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Three new widgets ("DOGOnews Current Events", "DOGObooks Book Reviews", and "DOGOmovies Movie Reviews") will be available under Appearance > Widgets, where you can add it to your sidebar and edit all settings of the widgets.
